# Project overview

Elite Schedule its a mobile app for adding your favourite teams and tracking their results in tournaments.
You can test this app in your computer but press f12 and turn on mobile version

## Technologies
- Angular 12.0.1
- Typescript 4.2.4
- ionic/angular 5.5.2
- ionic-native/ 5.34.0
- Firebase 6.1.4
